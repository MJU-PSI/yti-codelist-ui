Import codescheme

Import codescheme by uploading a CSV- or Excel formatted file. Look up more detailed guide and information about using codescheme-files from service help.

Using the imported file, a new codescheme is created or an older codescheme is update with new meta-information. Upon uploading the file, you get to review and cancel the changes if needed.



You can also add a new codescheme by creating it via the user interface of the Code List Service.
