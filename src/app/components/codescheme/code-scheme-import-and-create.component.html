Add code list

Import code list by uploading a CSV- or Excel formatted file. Look up more detailed guide and information about using code list files from service help.

Using the imported file, a new code list is created or an older code list is updated with new meta-information. Upon uploading the file, you get to review and cancel the changes if needed.



You can also add a new code list by creating it via the user interface of the Koodistot service.
