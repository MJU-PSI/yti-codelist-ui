Reference Data

The Reference Data module contains key common code lists used in public administration.

Code lists are standardised sets of codes and their associated values that are used to represent certain attributes in data sets. They ensure consistency and uniformity in the presentation of data across different systems and applications. Typically, code lists do not change often, except for occasional revisions. Code lists gain value when they are widely accepted and used. Uniform code lists enable:

Consistent presentation of data.

  • Consistent presentation of data.
  • Interoperability of systems.
  • Easier integration and data exchange between sectors and organizations.
  • Facilitate data exchange with external partners such as private companies, non-governmental or international organizations.
  • More effective data analysis.

The Reference Data module is directly linked to the Terminology and Data Models modules.

Šifranti

Modul Šifranti vsebuje ključne skupne šifrante, ki se uporabljajo v okviru javne uprave.

Šifranti so standardizirani nizi šifer in z njimi povezanih vrednosti, ki se uporabljajo za predstavitev določenih atributov v nizih podatkov.
Zagotavljajo doslednost in enotnost pri predstavitvi podatkov v različnih sistemih in aplikacijah. Šifranti se običajno ne spreminjajo pogosto, razen ob občasnih revizijah.
Šifranti pridobijo svojo vrednost, kadar so široko sprejeti in uporabljani.

Enotni šifranti omogočajo:

  • Dosledno predstavitev podatkov,
  • Interoperabilnost sistemov,
  • Lažjo integracijo in medresorsko izmenjavo podatkov,
  • Lažjo izmenjavo podatkov z zunanjimi partnerji kot so zasebna podjetja, nevladne ali mednarodne organizacije,
  • Učinkovitejšo analizo podatkov.

Modul šifranti se neposredno povezuje z moduloma Terminologija in Podatkovni modeli.
